# airlock

Airlock is an agent-action safety SDK: it wraps an agent's tool calls to gate
irreversible actions behind human approval, commit each side effect exactly
once via a durable ledger, and record everything in a tamper-evident,
hash-chained audit trail — so you never cause a side effect more than once and
can always prove what happened.

**Status: pre-alpha, under construction.** Nothing here is usable yet.
